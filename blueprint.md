# BAZI - 사주와 별자리 운세 서비스

## 프로젝트 개요

BAZI는 사용자의 사주 정보를 기반으로 동양의 전통적인 사주 명리학과 서양의 별자리 점성술을 융합하여 독창적이고 현대적인 운세 서비스를 제공하는 웹 애플리케이션입니다. 이 서비스는 사용자에게 깊이 있는 자기 이해와 삶의 방향성을 제시하는 것을 목표로 합니다.

## 주요 기능

*   **사주-별자리 퓨전 운세:** 사용자의 생년월일시를 기반으로 사주 팔자를 분석하고, 이를 서양 별자리와 결합하여 종합적인 운세 정보를 제공합니다.
*   **오늘의 운세:** 매일의 운세를 간략하게 확인할 수 있는 기능입니다.
*   **궁합 보기:** 다른 사용자와의 관계를 사주를 통해 분석하여 궁합 정보를 제공합니다.
*   **사주 초급:** 사주 명리학의 기본 개념을 설명하여 사용자의 이해를 돕습니다.
*   **12신:** 사주에 사용되는 12신에 대한 설명을 제공합니다.
*   **용어 사전:** 사주 관련 전문 용어를 쉽게 찾아볼 수 있는 사전을 제공합니다.

## 기술 스택

*   **프레임워크:** 없음 (No Framework)
*   **UI:** HTML, CSS, JavaScript (Web Components)
*   **디자인:** 모던 CSS (Container Queries, Cascade Layers, :has(), Logical Properties, Modern Color Spaces, CSS Variables)
*   **인프라:** Firebase Hosting

## 디자인 및 스타일

*   **시각적 컨셉:** 전체적으로 어두운 남색과 보라색 계열의 배경에 금색과 흰색 텍스트를 사용하여 신비롭고 고급스러운 분위기를 연출했습니다.
*   **타이포그래피:** 가독성을 고려한 폰트와 계층적인 텍스트 크기를 사용하여 정보의 중요도를 명확하게 전달합니다.
*   **레이아웃:** CSS Grid와 Flexbox를 사용하여 콘텐츠 섹션을 명확하게 구분하고 시각적으로 균형 잡힌 레이아웃을 구현했습니다.
*   **애니메이션:** 부드러운 전환 효과와 호버 효과를 추가하여 사용자 인터랙션을 향상시켰습니다.
*   **통합된 네비게이션 및 푸터:** 모든 페이지에 일관된 헤더와 푸터를 적용하여 사용자 경험을 통일하고, 주요 메뉴(`홈`, `사주초급`, `궁합보기`, `오늘의 운세`, `사주&별자리`, `용어사전`)와 법적 고지(`이용약관`, `개인정보처리방침`, `문의하기`)로의 접근성을 높였습니다.

## 프로젝트 진행 상황

### 1-8단계 (요약)

*   프로젝트 기본 구조 생성, SEO 최적화, 콘텐츠 보호, 도메인 설정, `sitemap` 및 `robots.txt` 구성 등 초기 개발 단계 완료.
*   '사주-별자리 퓨전 운세' 기능 개발 및 관련 UI/UX 개선 완료.
*   개인정보처리방침 페이지의 내용 보강 및 법적 고지 사항 업데이트.
*   이용약관의 저작권 귀속 조항 명확화.
*   푸터의 저작권 연도를 2024년으로 통일.

### 현재 계획: 코드 리뷰 및 개선

*   **목표:** 프로젝트 전반의 코드 품질을 향상시키고, 발견된 오류를 수정하며, 사용자 경험을 개선합니다.
*   **세부 계획:**
    1.  `privacy.html`: 개인정보 처리 관련 내용을 최신 법규에 맞게 검토하고, Google AdSense 관련 조항을 명확히 합니다.
    2.  `terms.html`: 저작권 관련 조항을 다시 한번 확인하고, 모호한 부분을 수정합니다.
    3.  모든 페이지의 푸터에 있는 저작권 연도를 현재 연도(2024)로 통일합니다.
    4.  기타 발견되는 사소한 오류나 오탈자를 수정합니다.
