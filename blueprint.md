# 프로젝트 청사진: 사주팔자 웹사이트

## 개요

이 문서는 Firebase Studio 환경에서 개발된 사주팔자 웹 애플리케이션의 설계, 기능 및 구현 계획을 담고 있습니다. 이 프로젝트는 HTML, CSS, JavaScript를 기반으로 한 프레임워크 없는 정적 웹사이트로, 사용자가 사주팔자에 대한 정보를 얻고 자신의 사주를 간단히 계산해 볼 수 있는 기능을 제공합니다.

## 구현된 기능 및 디자인

### 핵심 기능

*   **사주 정보 제공:** 사주팔자의 기본 개념, 십이신살, 용어 사전 등 다양한 정보성 콘텐츠를 제공합니다.
*   **사주 계산기:** 사용자가 자신의 생년월일시를 입력하면 간략화된 사주팔자 결과를 보여주는 웹 컴포넌트(`saju-calculator`)를 구현했습니다.
*   **궁합 계산기 (UI):** `love-compatibility.html` 페이지에 두 사람의 생년월일시를 입력받는 궁합 계산기 UI를 추가했습니다. (기능 구현 예정)
*   **일일 운세 뷰어 (UI):** `daily-fortune.html` 페이지에 날짜를 선택하고 해당 날짜의 운세를 볼 수 있는 UI를 추가했습니다. (기능 구현 예정)
*   **반응형 웹 디자인:** 모든 페이지는 모바일, 태블릿, 데스크톱 등 다양한 화면 크기에서 최적화된 사용자 경험을 제공하도록 설계되었습니다.
*   **모바일 네비게이션:** 화면이 좁아지면 햄버거 메뉴 버튼이 나타나며, 이를 통해 메뉴를 열고 닫을 수 있습니다.
*   **Google Analytics 연동:** 모든 페이지에 구글 애널리틱스 태그를 추가하여 사용자 방문 데이터를 추적합니다.

### 디자인 및 스타일

*   **시각적 컨셉:** 전체적으로 어두운 남색과 보라색 계열의 배경에 금색과 흰색 텍스트를 사용하여 신비롭고 고급스러운 분위기를 연출했습니다.
*   **타이포그래피:** 가독성을 고려한 폰트와 계층적인 텍스트 크기를 사용하여 정보의 중요도를 명확하게 전달합니다.
*   **레이아웃:** CSS Grid와 Flexbox를 사용하여 콘텐츠 섹션을 명확하게 구분하고 시각적으로 균형 잡힌 레이아웃을 구현했습니다.
*   **애니메이션:** 부드러운 전환 효과와 호버 효과를 추가하여 사용자 인터랙션을 향상시켰습니다.

## 프로젝트 진행 상황

### 1단계: 프로젝트 구조 및 기본 페이지 생성 (완료)

*   `index.html`: 메인 페이지. 프로젝트의 개요와 핵심 콘텐츠 포함.
*   `saju-basics.html`: 사주 기초 지식 페이지.
*   `12-deities.html`: 십이신살 정보 페이지.
*   `glossary.html`: 사주 관련 용어 사전.
*   `love-compatibility.html`: 궁합 정보 페이지.
*   `daily-fortune.html`: 오늘의 운세 페이지.
*   `privacy.html`, `terms.html`, `contact.html`: 법적 고지 및 문의 페이지.
*   `style.css`: 전체 사이트의 스타일을 정의하는 CSS 파일.
*   `main.js`: JavaScript 로직 및 웹 컴포넌트 정의.
*   `blueprint.md`: 프로젝트의 설계 및 계획을 기록하는 청사진 파일.

### 2단계: 헤더 및 네비게이션 일관성 확보 (완료)

*   모든 페이지의 헤더 구조를 일관되게 수정하고, 모바일 환경을 위한 반응형 네비게이션 기능을 구현했습니다.

### 3단계: 콘텐츠 정리 및 기능 UI 추가 (완료)

*   **블로그 페이지 제거:** 불필요한 `blog.html` 파일을 삭제하고, 모든 페이지의 네비게이션 메뉴에서 '블로그' 링크를 일괄적으로 제거하여 메뉴 구조를 단순화했습니다.
*   **궁합 계산기 UI 추가:** `love-compatibility.html` 페이지에 궁합 계산기 웹 컴포넌트(`<love-compatibility-calculator>`)를 배치하여 기능 구현을 위한 기반을 마련했습니다.
*   **일일 운세 UI 개선:** `daily-fortune.html` 페이지를 정적인 띠별 운세 목록에서, 날짜를 선택할 수 있는 입력 필드와 운세 결과를 보여줄 웹 컴포넌트(`<daily-fortune-viewer>`)를 포함하는 동적인 구조로 변경했습니다.
*   **웹 컴포넌트 정의:** `main.js`에 `LoveCompatibilityCalculator`와 `DailyFortuneViewer` 클래스를 추가하여 새로운 기능들의 기본 구조를 정의했습니다.

### 4단계: 핵심 기능 로직 구현 (진행 예정)

*   **궁합 계산 로직 개발:** `LoveCompatibilityCalculator` 웹 컴포넌트 내부에 두 사람의 사주 정보를 받아 궁합을 분석하고 결과를 표시하는 JavaScript 로직을 구현합니다.
*   **일일 운세 로직 개발:** `DailyFortuneViewer` 웹 컴포넌트가 날짜 선택 이벤트에 따라 해당 날짜의 띠별 운세 데이터를 동적으로 불러와 표시하는 기능을 구현합니다.
*   **사주 계산기 로직 고도화:** 현재 단순화된 `SajuCalculator`의 사주 계산 로직을 더 정확한 만세력 기반의 알고리즘으로 개선합니다.
