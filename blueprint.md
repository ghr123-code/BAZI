# Blueprint: 사주팔자 (Saju Palja) 웹 애플리케이션

## 1. 개요 (Overview)

본 프로젝트의 목표는 사용자에게 사주팔자에 대한 깊이 있는 정보와 개인화된 경험을 제공하는 고품질 웹 애플리케이션을 구축하는 것입니다. Google 애드센스 승인 기준을 충족하기 위해, 단순한 정보 제공을 넘어 풍부하고 독창적인 콘텐츠, 뛰어난 사용자 경험, 그리고 명확한 정보 구조를 갖추는 데 중점을 둡니다.

## 2. 프로젝트 목표 및 기능 (Goals & Features)

### 핵심 콘텐츠 (Core Content) - 애드센스 승인을 위한 깊이
- **사주팔자의 역사:** 고대 중국에서의 기원부터 한국 문화에 정착하기까지의 과정을 상세히 서술합니다.
- **사주팔자의 철학적 기반:** 과학적 근거보다는 음양오행설(陰陽五行說)과 천간(天干), 지지(地支) 등 사주를 구성하는 핵심 이론과 철학을 깊이 있게 다룹니다. 이는 오해의 소지를 없애고 콘텐츠의 질을 높입니다.
- **사주팔자 구성 요소 해설:**
    - 년주(年柱), 월주(月柱), 일주(日柱), 시주(時柱)의 의미와 역할
    - 십간(十干)과 십이지(十二支)의 상세한 설명
    - 대운(大運)의 개념과 인생 주기에 미치는 영향
- **현대적 해석:** 사주팔자를 현대 심리학이나 성격 분석과 연결하여 사용자가 자신을 더 잘 이해하는 도구로 활용할 수 있도록 안내합니다.

### 핵심 기능 (Core Functionality)
- **나의 사주팔자 계산기:**
    - 사용자가 자신의 양력 생년월일시를 입력합니다.
    - 입력된 정보를 바탕으로 만세력을 계산하여 여덟 글자(八字)를 표시합니다.
    - **(초기 버전)**: 프론트엔드에서 간단한 로직으로 결과를 보여주는 형태로 구현합니다. 정확한 만세력 변환은 복잡하므로, 초기에는 UI와 기본 결과 표시에 집중하고, 추후 라이브러리나 API 연동을 통해 고도화할 계획을 명시합니다.
    - 각 글자(간지)에 대한 기본적인 설명을 제공합니다.

### 필수 페이지 (Required Pages for AdSense)
- **개인정보 처리방침 (Privacy Policy):** 사용자의 데이터(생년월일)를 어떻게 처리하는지 명확히 밝힙니다.
- **서비스 이용약관 (Terms of Service):** 사이트 이용에 대한 규칙을 명시합니다.
- **문의하기 (Contact Us):** 사용자가 피드백을 제공할 수 있는 창구를 마련합니다.

## 3. 디자인 및 사용자 경험 (Design & UX)

- **시각적 컨셉:** 전통적인 동양 철학의 신비로움과 현대적인 미니멀리즘을 결합합니다. 깊은 남색, 검은색을 배경으로 금색, 흰색을 포인트 컬러로 사용하여 고급스럽고 신뢰감 있는 인상을 줍니다.
- **레이아웃:**
    - 모바일 우선 반응형 디자인을 적용하여 모든 기기에서 최적의 경험을 제공합니다.
    - 콘텐츠 가독성을 높이기 위해 충분한 여백과 명확한 타이포그래피 계층을 사용합니다.
- **타이포그래피:** 본문은 가독성이 좋은 `Noto Sans KR`과 같은 산세리프 폰트를, 제목이나 강조 문구에는 전통적인 느낌을 주는 `본명조`와 같은 세리프 폰트를 혼용하여 사용합니다.
- **인터랙션:**
    - 스크롤에 따른 부드러운 애니메이션 효과를 적용하여 콘텐츠에 집중할 수 있도록 합니다.
    - 사주 계산기 결과가 나타날 때 시각적인 효과를 주어 사용자의 흥미를 유발합니다.
    - 아이콘을 활용하여 정보를 직관적으로 이해할 수 있도록 돕습니다.

## 4. 기술 스택 (Technology Stack)

- **프론트엔드:**
    - **HTML, CSS, JavaScript (ES Modules):** 외부 프레임워크 없이 웹 표준 기술을 중심으로 구축합니다.
    - **Web Components:** '사주 계산기'와 같이 재사용 가능한 UI 요소를 캡슐화하여 코드의 모듈성과 유지보수성을 높입니다.
- **의존성 관리:** 서드파티 라이브러리는 가급적 CDN을 통해 추가하고, Subresource Integrity (SRI)를 사용하여 보안을 강화합니다.

## 5. 개발 계획 (Current Plan)

1.  **`blueprint.md` 작성:** 현재 단계. 프로젝트의 청사진을 정의합니다.
2.  **`index.html` 구조 설계:** 콘텐츠 섹션(역사, 철학, 계산기 등)과 푸터(개인정보처리방침 등 링크)의 기본 마크업을 작성합니다.
3.  **`style.css` 기본 스타일링:** 전체적인 색상, 폰트, 레이아웃 등 기본 디자인 시스템을 구축합니다.
4.  **콘텐츠 추가:** `index.html`에 사주팔자의 역사, 철학 등 텍스트 콘텐츠를 채워 넣습니다.
5.  **Web Component 개발 (`main.js`):** `<saju-calculator>` 커스텀 엘리먼트를 생성합니다. 생년월일시 입력 UI와 결과 표시 영역을 포함하며, 초기에는 간단한 예시 결과를 반환하는 로직을 구현합니다.
6.  **`index.html`에 스크립트 및 컴포넌트 연동:** `style.css`와 `main.js` 파일을 `index.html`에 연결하고, 생성한 `<saju-calculator>`를 본문에 배치합니다.
7.  **최종 검토 및 수정:** 전체적인 디자인과 기능을 검토하고, 오류를 수정하며 완성도를 높입니다.
