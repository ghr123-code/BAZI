# 프로젝트 청사진: 사주팔자 웹사이트

## 개요

이 문서는 Firebase Studio 환경에서 개발된 사주팔자 웹 애플리케이션의 설계, 기능 및 구현 계획을 담고 있습니다. 이 프로젝트는 HTML, CSS, JavaScript를 기반으로 한 프레임워크 없는 정적 웹사이트로, 사용자가 사주팔자에 대한 정보를 얻고 자신의 사주를 간단히 계산해 볼 수 있는 기능을 제공합니다.

## 구현된 기능 및 디자인

### 핵심 기능

*   **사주 정보 제공:** 사주팔자의 기본 개념, 역사, 철학, 현대적 해석 등 다양한 정보성 콘텐츠를 제공합니다.
*   **사주 계산기:** 사용자가 자신의 생년월일시를 입력하면 간략화된 사주팔자 결과를 보여주는 웹 컴포넌트(`saju-calculator`)를 구현했습니다.
*   **반응형 웹 디자인:** 모든 페이지는 모바일, 태블릿, 데스크톱 등 다양한 화면 크기에서 최적화된 사용자 경험을 제공하도록 설계되었습니다.
*   **모바일 네비게이션:** 화면이 좁아지면 햄버거 메뉴 버튼이 나타나며, 이를 통해 메뉴를 열고 닫을 수 있습니다.

### 디자인 및 스타일

*   **시각적 컨셉:** 전체적으로 어두운 남색과 보라색 계열의 배경에 금색과 흰색 텍스트를 사용하여 신비롭고 고급스러운 분위기를 연출했습니다.
*   **타이포그래피:** 가독성을 고려한 폰트와 계층적인 텍스트 크기를 사용하여 정보의 중요도를 명확하게 전달합니다.
*   **레이아웃:** CSS Grid와 Flexbox를 사용하여 콘텐츠 섹션을 명확하게 구분하고 시각적으로 균형 잡힌 레이아웃을 구현했습니다.
*   **애니메이션:** 부드러운 전환 효과와 호버 효과를 추가하여 사용자 인터랙션을 향상시켰습니다.

## 현재 계획 및 단계

### 1단계: 프로젝트 구조 및 기본 페이지 생성 (완료)

*   `index.html`: 메인 페이지. 프로젝트의 개요와 핵심 콘텐츠 포함.
*   `saju-basics.html`: 사주 기초 지식 페이지.
*   `12-deities.html`: 십이신살 정보 페이지.
*   `glossary.html`: 사주 관련 용어 사전.
*   `love-compatibility.html`: 궁합 정보 페이지.
*   `daily-fortune.html`: 오늘의 운세 페이지.
*   `blog.html`: 블로그 목록 페이지.
*   `privacy.html`, `terms.html`, `contact.html`: 법적 고지 및 문의 페이지.
*   `style.css`: 전체 사이트의 스타일을 정의하는 CSS 파일.
*   `main.js`: 사주 계산기 웹 컴포넌트 및 모바일 네비게이션 로직을 포함하는 JavaScript 파일.
*   `blueprint.md`: 프로젝트의 설계 및 계획을 기록하는 청사진 파일.

### 2단계: 헤더 및 네비게이션 일관성 확보 (완료)

*   **문제:** 각 HTML 파일의 헤더 구조가 일관되지 않고, 모바일 대응이 미흡했습니다.
*   **해결:**
    1.  모든 HTML 파일의 `<header>` 구조를 `index.html`과 동일하게 수정하여 일관성을 확보했습니다.
    2.  `h1` 태그와 `nav`를 포함하는 명확한 구조로 변경하고, 모바일 화면을 위한 햄버거 메뉴 버튼 (`<button class="nav-toggle">`)을 추가했습니다.
    3.  모든 페이지에 구글 애널리틱스 추적 코드를 삽입하여 사이트 이용 통계를 수집할 수 있도록 기반을 마련했습니다.
    4.  `main.js`를 불러오는 `<script>` 태그에 `type="module"`과 `defer` 속성을 추가하여 최신 JavaScript 모듈 시스템을 사용하고 페이지 로딩 성능을 최적화했습니다.
    5.  `main.js` 파일에 모바일 네비게이션 토글 기능을 구현하여 햄버거 메뉴가 실제로 동작하도록 했습니다.
