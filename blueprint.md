# 프로젝트 청사진: 사주팔자 웹사이트

## 개요

이 문서는 Firebase Studio 환경에서 개발된 사주팔자 웹 애플리케케이션의 설계, 기능 및 구현 계획을 담고 있습니다. 이 프로젝트는 HTML, CSS, JavaScript를 기반으로 한 프레임워크 없는 정적 웹사이트로, 사용자가 사주팔자에 대한 정보를 얻고 자신의 운세를 다각도로 분석해 볼 수 있는 기능을 제공합니다.

## 구현된 기능 및 디자인

### 핵심 기능

*   **사주 정보 제공:** 사주팔자의 기본 개념, 십이신살, 용어 사전 등 다양한 정보성 콘텐츠를 제공합니다.
*   **사주 계산기:** 사용자가 자신의 생년월일시를 입력하면 간략화된 사주팔자 결과를 보여주는 웹 컴포넌트(`saju-calculator`)를 구현했습니다.
*   **궁합 계산기 (UI):** `love-compatibility.html` 페이지에 두 사람의 생년월일시를 입력받는 궁합 계산기 UI를 추가했습니다. (기능 구현 예정)
*   **일일 운세 뷰어 (UI):** `daily-fortune.html` 페이지에 날짜를 선택하고 해당 날짜의 운세를 볼 수 있는 UI를 추가했습니다. (기능 구현 예정)
*   **퓨전 운세 기능:** `astrology-fusion.html` 페이지를 신설하고, 동양의 사주와 서양의 별자리를 결합하여 사용자의 성향을 입체적으로 분석해주는 '퓨전 운세' 기능을 구현했습니다.
    *   **웹 컴포넌트:** `FusionAstrologyResult` 웹 컴포넌트를 개발하여 사주와 별자리 결과를 동적으로 생성하고 시각적으로 표현합니다.
    *   **쉬운 해석:** 전문 용어 대신 일상적인 언어를 사용하여 핵심 성향과 별자리 특성을 쉽게 이해할 수 있도록 했습니다.
*   **반응형 웹 디자인:** 모든 페이지는 모바일, 태블릿, 데스크톱 등 다양한 화면 크기에서 최적화된 사용자 경험을 제공하도록 설계되었습니다.
*   **SEO 및 기술 표준:** 모든 페이지에 시맨틱 HTML, 메타 태그, `robots.txt`, `sitemap.xml` 등을 적용하여 검색 엔진 최적화를 수행했습니다.
*   **분석 및 광고:** 구글 애널리틱스 및 애드센스 태그 자리표시자를 포함하여 데이터 분석 및 수익화 기반을 마련했습니다.

### 디자인 및 스타일

*   **시각적 컨셉:** 전체적으로 어두운 남색과 보라색 계열의 배경에 금색과 흰색 텍스트를 사용하여 신비롭고 고급스러운 분위기를 연출했습니다.
*   **타이포그래피:** 가독성을 고려한 폰트와 계층적인 텍스트 크기를 사용하여 정보의 중요도를 명확하게 전달합니다.
*   **레이아웃:** CSS Grid와 Flexbox를 사용하여 콘텐츠 섹션을 명확하게 구분하고 시각적으로 균형 잡힌 레이아웃을 구현했습니다.
*   **애니메이션:** 부드러운 전환 효과와 호버 효과를 추가하여 사용자 인터랙션을 향상시켰습니다.

## 프로젝트 진행 상황

### 1-7단계 (요약)
*   프로젝트 기본 구조 생성, SEO 최적화, 콘텐츠 보호, 도메인 설정, `sitemap` 및 `robots.txt` 구성 등 초기 개발 단계 완료.

### 8단계: 사주-별자리 퓨전 운세 기능 개발 (완료)
*   **신규 페이지 생성:** `astrology-fusion.html` 파일을 생성하고 SEO 및 소셜 공유 태그, 애널리틱스/애드센스 자리표시자를 포함하여 기본 구조를 완성했습니다.
*   **웹 컴포넌트 개발:** 사용자의 생년월일시를 기반으로 사주팔자와 서양 별자리를 계산하고, 그 결과를 결합하여 보여주는 `FusionAstrologyResult` 커스텀 웹 컴포넌트를 `main.js`에 구현했습니다.
*   **UI/UX 개선:** `style.css`를 수정하여 결과 카드의 디자인을 개선하고, 아이콘을 추가하여 가독성과 시각적 매력을 높였습니다.
*   **사이트맵 업데이트:** `sitemap.xml`에 `astrology-fusion.html` 페이지를 추가하여 검색 엔진이 새로운 콘텐츠를 빠르게 인덱싱할 수 있도록 조치했습니다.

### 9단계: 기능 로직 구현 (진행 예정)
*   **궁합 계산 로직 개발:** `LoveCompatibilityCalculator` 웹 컴포넌트 내부에 두 사람의 사주 정보를 받아 궁합을 분석하고 결과를 표시하는 JavaScript 로직을 구현합니다.
*   **일일 운세 로जिक 개발:** `DailyFortuneViewer` 웹 컴포넌트가 날짜 선택 이벤트에 따라 해당 날짜의 띠별 운세 데이터를 동적으로 불러와 표시하는 기능을 구현합니다.
*   **사주 계산기 로직 고도화:** 현재 단순화된 `SajuCalculator`의 사주 계산 로직을 더 정확한 만세력 기반의 알고리즘으로 개선합니다.
